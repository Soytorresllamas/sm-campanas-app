# Plan 26-27 · marketing y servicios académicos

## Servicios académicos — tres perfiles

Tres etapas de servicio: **uso**, **profundización** y **didácticas específicas** (extra-curriculares; p. ej. talleres de liderazgo).

- **SMART:** uso → profundización → didácticas específicas (dentro de 26-27).
- **CORE:** didácticas específicas (entrada, primavera 2027) → uso → profundización (jun-jul-ago 2027).

### Modelo de acompañamiento

| Etapa | Quién entrega |
|---|---|
| Uso | Asesor pedagógico empleado de SM (KAM) |
| Profundización | Asesor SM o externo bajo demanda |
| Didácticas específicas | Especialistas externos |

## Coberturas y capacidad

Talleres por colegio: uso 1-3, profundización 2-3, didácticas específicas 1. Con 10 asesores empleados (2 talleres/día × 5 días ≈ 433/mes), uso y profundización se cubren casi íntegramente internamente; las didácticas específicas y el sobrecupo se cubren con externos. Ver el **Simulador** para mover las variables.

## Retención vs conquista

La cobertura mensual se parte en clientes retenidos y de conquista. Retención base: SMART 94%, CORE 89%. La conquista se concentra en CORE (primavera).
